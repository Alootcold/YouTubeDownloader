# 更新日志

## v2.0.0 - 换新版 (2026年3月27日)

> 全新版本，重磅来袭

### 技术重构

本次更新是一次全面重构，告别过去的版本，带来全新的体验：

- **编程语言**：从原有语言迁移至 C#，提升开发效率与代码安全性
- **UI 框架**：采用 WinUI 3 (Windows App SDK)，打造现代化 Windows 原生界面
- **.NET 版本**：基于 .NET 8.0，享受最新运行时性能优化
- **架构升级**：采用 MVVM 架构模式，代码结构更清晰、易维护
- **未来布局**：为后续功能扩展奠定坚实基础

### 新增功能

- 全新设计的 WinUI 3 用户界面

### 功能优化

- 优化视频解析算法，提升稳定性
- 改进下载进度显示，实时反馈
- 增强错误处理机制

### 用户体验

## v1.0.0 - 初始版本 (2025年3月26日)

### 新增功能

- 支持下载普通 YouTube 视频（横屏）
- 支持下载 YouTube Shorts 视频（竖屏）
- 实时显示下载进度和百分比
- 视频信息预览功能，包括缩略图、标题、上传者、时长等
- 支持深色/浅色主题切换
- 自动保存用户设置和偏好
- 支持选择保存位置

### 技术特性

- 基于 PyQt5 构建的美观用户界面
- 使用 yt-dlp 作为下载引擎，支持最高分辨率视频下载
- 多线程下载，不阻塞主界面
- 平滑的下载进度显示
- 错误处理机制，提供用户友好的体验
- 支持 Windows 系统

### 安装方式

- 提供便捷的安装程序（YouTubeDownloader_Setup.exe）
- 支持便携式使用（YouTubeDownloader/YouTubeDownloader.exe）
- 支持从源代码构建（python setup.py build）
