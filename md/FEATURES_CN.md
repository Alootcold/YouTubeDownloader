## 核心功能

### 视频下载
- 只需输入 URL 即可下载 YouTube 视频
- 支持标准链接和短链接（youtu.be）

### 画质选择
- 多种画质可选：360p、480p、720p、1080p
- 自动选择最佳可用画质
- 视频音频自动合并

### 简洁界面
- 简洁现代的 WinUI 3 设计
- 侧边栏导航，操作便捷
- 实时下载进度跟踪
- 下载历史记录管理

### 内置工具
- 内置 yt-dlp 引擎，解析可靠
- 内置 ffmpeg 视频处理
- 无需额外安装软件

### 隐私保护
- 不收集任何个人数据
- 所有处理在本地完成
- 下载文件保存在您选择的位置

---

## 系统要求

- Windows 10（版本 1809）或更高版本
- Windows 11
- 网络连接（用于下载视频）
