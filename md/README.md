# YouTube视频下载器 换新版

一款简洁高效的 YouTube 视频下载工具，支持多种格式和画质选择。

## 功能特点

- 支持 YouTube 视频下载
- 多种画质和格式选择
- 内置 ffmpeg 视频处理
- 实时下载进度显示
- 下载历史记录管理
- 简洁直观的用户界面

## 系统要求

- Windows 10 (版本 1809) 或更高版本
- Windows 11

## 使用说明

1. 在首页输入 YouTube 视频链接
2. 选择 desired 画质
3. 点击下载按钮
4. 在下载列表查看进度
5. 下载完成后可在下载历史中找到记录

## 技术架构

- WinUI 3
- .NET 8.0
- yt-dlp 视频解析引擎
- ffmpeg 视频处理
